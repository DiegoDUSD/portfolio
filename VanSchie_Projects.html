Name: National Breakdown
Date: June 2017 - September 2017
Course: Personal project
Objective: This application is a personal project made by me, and was made to keep track of information regarding a country's diplomacy, their economy, etc. Essentially, this application will save profiles storing certain information to a file, and these profiles can be saved, changed, and loaded. This also is able to put this information together, and generates a readable text-format report. Finally, this application also has support for writing new windows that may be opened at certain times using CINI(Complex-INI), a library also made by me.
